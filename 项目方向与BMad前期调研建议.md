## 当前主线固定模板

- 主线名：`本地服务商家预约爽约挽回助手`
- 推荐理由：`用户规模更大`，覆盖 `美业 / 牙科 / 私教 / 培训 / 摄影 / 家政 / 本地维修` 等大量预约型本地服务商家
- 推荐理由：`一旦痛就愿意立刻付钱`，因为爽约直接对应 `空档时段收入损失`
- 推荐理由：`高频`，只要商家持续接预约，这个问题就会反复发生
- 推荐理由：`更可持续`，不是一次性工具，而是可长期承载 `确认 / 提醒 / 补位 / 损失统计`
- 推荐理由：`适合网站首发`，可以先做 `不替代原预约系统` 的外挂式产品，降低首版复杂度
- 首版功能：`预约确认`
- 首版功能：`多轮提醒`
- 首版功能：`爽约风险识别`
- 首版功能：`候补补位`
- 首版功能：`损失统计`
- 不做什么：`不替代商家现有预约系统`，当前先做外挂式挽回工具
- 不做什么：`不一开始做全套预约 SaaS / POS / CRM`
- 不做什么：`不同时覆盖过多垂直行业的深度定制流程`
- 下一步验证：`做中国大陆 + 海外竞品分析，确认这个问题是否已被成熟产品充分覆盖`
- 下一步验证：`比较美业 / 牙科 / 私教 / 培训 / 摄影 / 家政 / 本地维修，找最适合首发的单一垂直切口`
- 下一步验证：`判断网站首发是否仍成立，还是需要调整为更适合的单一终端`
